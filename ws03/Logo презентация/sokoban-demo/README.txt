Установить Logo
Запустить файл project из папки sokoban
Запуск main из командной строки
Для управления необходимо переключиться (например, кликом мыши) на игровое поле
Клавиша W - вверх
Клавиша S - вниз
Клавиша A - влево
Клавиша D - вправо
Клавиша R - пройти уровень заново
При возникновении ошибки убедиться, что все файлы уровней присутствуют в папке
Результаты прохождения сохраняются в файл result.txt
